# ⚠️ 重要提示：请不要手动关闭浏览器窗口！

## 正确使用方式

当您运行程序时：

```bash
python ddg_fetcher_browser.py --vendor ti --keywords "LM5164" --max-results 2 --out downloads
```

您会看到：
1. ✅ Chrome 浏览器自动打开
2. ✅ 自动访问 DuckDuckGo 搜索
3. ✅ 自动打开 PDF 链接
4. ⏳ 等待下载完成（最多3分钟）

## ❌ 不要做

- **不要手动关闭浏览器窗口！**
- 不要点击浏览器的 X 按钮
- 不要按 Alt+F4 关闭浏览器

## ✅ 应该做

- **让浏览器窗口保持打开**
- 让程序自动完成所有操作
- 程序完成后会自动关闭浏览器
- 您可以在终端按 Ctrl+C 来中止程序（程序会正确清理）

## 🔍 如何判断程序在运行

查看终端输出：
- `⏳ 等待下载完成（超时: 180秒）...` - 正在等待
- `  下载中: xxx.pdf.crdownload (123 KB) - 15秒` - 正在下载
- `✓ 下载完成: xxx.pdf (456 KB)` - 下载成功

## 💡 提示

如果等待时间太长（超过30秒还没下载），可能是：
1. TI 网站响应慢
2. PDF 文件较大
3. 网络连接慢

**耐心等待即可，不要关闭浏览器！**
